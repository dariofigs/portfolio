Description

While the team was at an offsite, Matt and I worked together to create an internal UI Kit for the team to use on client engagements.

Context

When working with clients, each designer had their preferred UI Kit to use which led to clients being unintentionally biased towards screens based on the kit used. We hoped the kit would remove that bias by everyone using the same system as well as speeding up people's workflows by updating it with whatever they needed for the client they were on.

UI Kit

Built a UI Kit that the team used internally to quickly iterate on client design work and increase designer efficiency and cohesion and reduce designer workload.
